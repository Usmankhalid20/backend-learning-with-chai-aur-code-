2: Why Nodemon?
answer: Nodemon is a tool that automatically restarts the Node.js application when it detects any changes in the code. This helps developers to quickly test and see the changes they made in their code without having to manually restart the server each time.

3: Installing Express@4
answer: npm install express@4

4: Installing Nodemon
answer: npm install -g nodemon

4: Request parameters and queries
answer: Request parameters are passed as an object to the route handler function, while query parameters are passed as an object to the req.query property.

5: Static Files
answer: Express.js provides a built-in middleware function called express.static() that can be used to server static files from a directory.
